# End-to-End DevSecOps CI/CD Pipeline

## Project Overview

This project demonstrates the implementation of a fully automated DevSecOps CI/CD pipeline for a Spring Boot application. The pipeline integrates security scanning, code quality analysis, containerization, and automated deployment on AWS EC2.

## Tech Stack

* GitHub
* Jenkins
* Maven
* SonarQube
* OWASP Dependency Check
* Trivy
* Docker
* Docker Compose
* Spring Boot
* MySQL
* AWS EC2
* Linux

## Pipeline Workflow

1. Source Code Checkout from GitHub
2. Maven Build & Package
3. SonarQube Static Code Analysis
4. Quality Gate Validation
5. OWASP Dependency Check Scan
6. Trivy File System Scan
7. Docker Image Build
8. Trivy Image Vulnerability Scan
9. Docker Compose Deployment
10. Application Deployment on AWS EC2

## Security Features

* Static Application Security Testing (SAST)
* Dependency Vulnerability Scanning
* File System Security Scanning
* Docker Image Security Scanning
* Automated Security Reporting
* Shift-Left Security Implementation

## Key Achievements

* Automated End-to-End CI/CD Pipeline
* Secure Containerized Deployment
* Early Vulnerability Detection
* Improved Code Quality
* Automated Security Validation
* Faster and Reliable Releases

## Project Architecture

GitHub → Jenkins → Maven Build → SonarQube Analysis → Quality Gate → OWASP Scan → Trivy FS Scan → Docker Build → Trivy Image Scan → Docker Compose → AWS EC2 Deployment
